# 上海爱斯利嘉半导体科技有限公司官网

## 项目简介
这是上海爱斯利嘉半导体科技有限公司的官方网站项目，采用现代化的响应式设计，为访问者提供良好的浏览体验。

## 技术栈
- HTML5
- CSS3
- JavaScript
- 响应式设计

## 主要功能
- 首页：公司简介和核心数据展示
- 关于我们：公司历史、核心价值观和主要成就
- 解决方案：各类技术解决方案展示
- 服务：技术支持、软件开发、咨询顾问等服务介绍
- 联系我们：联系方式和在线留言表单
